# casefold()和lower()
#python/string函数 

---
**作用**：将字符全部小写

### 使用方法
```python
newstr = string.casefold();			

newstr = string.lower();	//都是将字符串变成小写传给newstr
```

####  参数
>没有参数

#### 实例
```python
name = 'DEWEI';
new_name = name.lower()
print(new_name)	//输出结果：dewei
```

`casefold()`和`lower()`的使用方式都是一样的。
