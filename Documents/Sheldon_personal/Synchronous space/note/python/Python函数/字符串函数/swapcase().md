# swapcase()
#python/string函数

---

**功能：**：字符串大小写进行转换，大变小，小变大

### 使用方式：
```python
newstr = string.swapcase()		//将string的字符串大小写进行转换，大变小，小变大传给newstr
```
参数：
>函数括弧内不能传入参数

实例：
```python
name = 'xiaoMu';
new_name=name.swapcase();
print(new_name);		//输出结果：XIAOmU
```
